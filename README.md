# rodrigoschmitt.github.io

Personal resume site — static multi-page HTML/CSS/JS, no build step. Served via GitHub Pages from the repo root.

Pages: `index.html` (Home), `work.html`, `vaulto.html`, `about.html`, `contact.html` — all share `css/styles.css` and `js/scripts.js`.

Preview locally with `python3 -m http.server 8000`.
